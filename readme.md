# لوحة التحكم Dashboard

واجهة لوحة تحكم حديثة تم تصميمها باستخدام HTML و CSS فقط، مع تصميم عصري داكن يحتوي على بطاقات إحصائية وجدول منتجات وشريط جانبي تفاعلي.

---

# المميزات

- تصميم حديث Dark UI
- استخدام CSS Variables لتنظيم الألوان
- Responsive Design لجميع الشاشات
- Sidebar جانبي للتنقل
- بطاقات إحصائية للمعلومات
- جدول منتجات منسق
- تأثيرات Hover وانتقالات سلسة
- بدون أي مكتبات خارجية
